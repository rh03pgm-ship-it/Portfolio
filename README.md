# インフラエンジニア ポートフォリオ

## About Me

インフラエンジニアとして11年以上、医療機関向け自社パッケージシステムの導入・構築・運用保守に従事してきました。

Windows Server、Active Directory、SQL Serverを中心に、サーバ・ネットワークの設計、構築、運用、障害対応まで幅広く担当しています。

また、YAMAHAルータを用いたネットワーク構築やVPN環境構築、Webシステム運用、自動化ツール開発などにも携わってきました。

現在はAzureを中心としたクラウド技術を学習しており、オンプレミスとクラウド双方に対応できるインフラエンジニアを目指しています。
