# インフラエンジニア ポートフォリオ

## 自己紹介

インフラエンジニアとして11年以上、医療機関向け自社パッケージシステムの導入・構築・運用保守に従事してきました。

Windows Server、Active Directory、SQL Serverを中心に、サーバー・ネットワークの設計、構築、運用、障害対応まで幅広く担当しています。

また、YAMAHAルータを用いたネットワーク構築やVPN環境構築、Webシステム運用、自動化ツール開発などにも携わってきました。

現在はAzureを中心としたクラウド技術を学習しており、オンプレミスとクラウド双方に対応できるインフラエンジニアを目指しています。

## 03. インフラ運用自動化ツール開発

### 概要

顧客環境へのシステム導入および運用保守業務の効率化を目的として、Pythonを用いた自動化ツールを開発。

ソフトウェア動作に必要な環境設定やレジストリ変更の自動化に加え、障害発生時の設定修復やログ収集機能を実装し、導入作業および運用保守業務の効率化に貢献した。

### 担当業務

* Windowsレジストリ設定変更の自動化
* ソフトウェア動作に必要な環境設定の自動適用
* セットアップ作業の効率化および標準化
* 障害発生時の設定修復処理の実装

### 成果

* ソフトウェア導入時の環境構築作業を効率化
* セットアップ工数の削減による導入作業の迅速化
* 作業品質の標準化および手順ミスの低減
* 障害対応時間の短縮
