# インフラエンジニア ポートフォリオ

## 自己紹介

インフラエンジニアとして11年以上、医療機関向け自社パッケージシステムの導入・構築・運用保守に従事してきました。

Windows Server、Active Directory、SQL Serverを中心に、サーバー・ネットワークの設計、構築、運用、障害対応まで幅広く担当しています。

また、YAMAHAルータを用いたネットワーク構築やVPN環境構築、Webシステム運用、自動化ツール開発などにも携わってきました。

現在はAzureを中心としたクラウド技術を学習しており、オンプレミスとクラウド双方に対応できるインフラエンジニアを目指しています。

## 03. インフラ運用自動化ツール開発

### 概要

顧客環境へのシステム導入時に発生するWindows端末の初期セットアップ作業を効率化するため、Pythonを用いた自動化ツールを開発。

レジストリ設定変更や各種システム設定を自動化し、手作業による設定ミスの削減と作業品質の標準化を実現した。

### 担当業務

Windowsレジストリ設定変更の自動化
動作検証および運用手順の整備

### 使用技術

Python
Windows（レジストリ操作）
バッチファイル

### 成果

* セットアップ工数削減
* 作業品質の均一化
* 手順ミス低減
